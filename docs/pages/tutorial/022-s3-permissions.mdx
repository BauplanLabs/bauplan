---
title: "S3 Permissions Example"
description: "Example S3 bucket policy granting public GetObject and ListBucket access so Bauplan can read Parquet and CSV files when importing data in the sandbox."
---

```json
{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Sid": "PublicReadGetObject",
      "Effect": "Allow",
      "Principal": "*",
      "Action": "s3:GetObject",
      "Resource": "arn:aws:s3:::YOURBUCKET/*"
    },
    {
      "Sid": "PublicListBucket",
      "Effect": "Allow",
      "Principal": "*",
      "Action": "s3:ListBucket",
      "Resource": "arn:aws:s3:::YOURBUCKET"
    }
  ]
}
```
