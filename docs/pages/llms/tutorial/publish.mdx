

# Review and Publish

When you're satisfied with the pipeline results:

**Compare against main:**

```
Show me the diff between my pipeline branch and main. What new tables and data were created?
```

**Merge to production:**

```
The pipeline looks good. Merge this branch to main and clean up the dev branch.
```

The agent will:

1. Merge your pipeline code to `main`
2. Clean up the development branch
